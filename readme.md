### canvas 绘制雷达图

#### 使用说明
> 定义初始数组，数组长度大于3才会显示数据
> 例如： let arr = [10, 20, 30, 60]

#### 绘制说明
> 跟据数组中的最大值为依据，区间取整拾(50-60间则取60) 显示
